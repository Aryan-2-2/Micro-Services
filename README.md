# Micro-Services
## REST-API for Hotel-Rating-System in Micro-service architecture.


## Tech Stack
- Java
- Spring Boot with JPA
- Postgres
- Maven
- Lombok
- Fault Tolerance
- RETRY
- Rate Limiter
